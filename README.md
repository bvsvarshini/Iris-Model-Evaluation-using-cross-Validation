# 🌸 Iris Model Evaluation using 5-Fold Cross Validation

## 🎯 Objective
To evaluate the performance of a deep learning model on the Iris dataset using 5-Fold Cross Validation.

## 📚 Project Overview
This project uses a neural network built with TensorFlow/Keras and evaluates it using k-fold cross-validation.
